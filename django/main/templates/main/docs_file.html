File Management

Upload and download settings.

Only Professional tier users will be able to perform file uploads and downloads.

Upload

  1. Click to open the left side menu.
    • Left side menu location
  2. Click the "Filebrowser" icon.
    • Filebrowser Icon
  3. Click the "Downloads" folder to change to this folder location.
    Users are only allowed to upload/download to/from the Downloads folder.
    • Downloads Folder
  4. Upload file one of two ways.
    1. Click the "Upload Files" button.
    2. Drag and drop a file into the file browser window.
      • Upload File

Download

  1. Click to open the left side menu.
    • Left side menu location
  2. Click the "Filebrowser" icon.
    • Filebrowser Icon
  3. Click the "Downloads" folder to change to this folder location.
    Users are only allowed to upload/download to/from the Downloads folder.
    • Downloads Folder
  4. Click on desired file to download.
    Users may download single files or any file archive. However, downloading un-archived folders is not possible.
    By performing this action, you will be actively downloading files from the session into a non-password-protected ZIP file. These files are offered “as-is” and may contain malicious content. By downloading, you acknowledge and accept all risks, confirm you have the necessary knowledge to handle potentially harmful files, and agree that you are solely responsible for any resulting damage or loss.
